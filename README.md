# DC Project

This repository contains our data center networking project based on the paper *Dynamic and Load-Aware Flowlet for Load-Balancing in Data Center Networks*.

- Appaji Nagaraja Dheeraj - 241CS110

## Repository Layout

- `website/` - the interactive web application for concepts, simulation, and P4 code viewing
- `simulation/` - supporting simulation logic and related assets
- `Implementation Setup.md` - implementation notes
- `Architecture.md` - system architecture overview
- `Methodology.md` - experimental and simulation methodology
- `Results and explanation.md` - results summary and interpretation

## Paper Focus

The project compares ECMP, Flowlet, and DLAF for load balancing in data center networks. The website presents the concepts and the simulator visualizes how path selection, flowlet splitting, and load awareness affect balance across links.
